# SocialDog 朝昼夕 3投稿方針

投資家Fの文体をベースに、毎日同じ投稿に見えないようにするための投稿設計です。

## 1日の構成

| 時刻 | 役割 | 文体 |
| --- | --- | --- |
| 朝 | 注目ニュース | インプレッションを狙うニュース3選 |
| 昼 | 意味のある声かけ | 生活感、比喩、読者の行動、成長の一言 |
| 夕 | 夜の相場メモ | 指標時刻、ニュース、値動きのクセ、学びのメモ |

## 朝投稿の切り口

- ドル円と介入警戒
- GOLDさんの日足
- NASDAQとAI関連
- 日経平均と前場の需給
- 米指標前の市場心理
- 地政学ニュースとGOLD
- 週末前の見送り

## 朝投稿の固定方針

朝は注目ニュースを箇条書きする。
インプレッションを狙うため、固有名詞・数字・意外性・話題性が強いものを優先する。

```text
＼今日の注目ニュース🌈🐻／

朝から気になる材料が並んでますね...

✅OpenAI、AI半導体、NASDAQなど話題性のあるニュース
✅GOLD、ドル円、日経平均など数字があるニュース
✅米PPI、ECB、FOMCなど時間が決まっているニュース

数字そのものより、
ユーロ・米金利・GOLDの
どれが先に動いたかだけ残します😺

投資家Fより💌
```

選ぶ基準:

- 有名な固有名詞がある
- 数字がある
- 直近性がある
- 読者が反応しやすい
- 相場に関係する
- `え、それ何？` と読みたくなる

## タイトルのランダム化

昼と夕は、毎回同じタイトルにしない。
`＼お昼のひとこと／` と `＼NY前の点検／` を固定で使い続けると投稿が機械的に見えるため、候補から変える。

昼タイトル例:

- `＼お昼の小さな前進😺🌸／`
- `＼午後の一歩メモ🌷🐻／`
- `＼Fの昼メモ🥰🌈／`
- `＼午後に効くひとこと😺✨／`

夕タイトル例:

- `＼NY前のニュース整理🐻‍❄️🌈／`
- `＼夜のマーケットメモ😺🌙／`
- `＼21時台前のFメモ🌈🐻／`
- `＼NY前に見たい材料🐻‍❄️✨／`

## 昼投稿の切り口

- コーヒー
- 布団
- ランチ
- 散歩
- メモ帳
- ゲーム
- 猫
- 作業中の集中力
- 急上昇キーワードを日常の比喩にする

昼投稿は「ただのゆる投稿」にしない。
読んだ人のエネルギーが少し上がり、行動が1つ決まる文章にする。

必ず入れるもの:

- 読者ができる小さい行動
- 成長につながる一言
- Fらしいやわらかい比喩

例:

```text
＼お昼のひとこと😺🌸／

午前中にうまく進まなかった日ほど、
午後に全部取り返そうとしがちです。

でもFは、
そういう日は予定を1つ減らして
頭の余白を作ります🐻

メモ帳の一番上に、
今日やる1つだけを書きます🌷

投資家Fより💌
```

## 夕投稿の切り口

- NY前の指標時刻
- 指標前の市場心理
- 最初の値動きに反応しすぎない話
- 利益や損失が出た時の心の動き
- 金曜日の見送り
- 週末リスク
- 二発目の動きを見る話

夕投稿では、F自身の注文、ポジション、売買予定は書かない。
読者への投資助言にも見えないようにする。
書くのは「市場が何に反応しているか」「数字のどこを見るか」「今日の学びをどう残すか」まで。
締めは抽象的にしない。
時間、対象、手元の動作が見える言葉で終わる。

よい締め:

- `21:30後の最初の15分は、米金利・GOLD・NASDAQの順番だけ見ます`
- `メモ帳の一番上に、今日やる1つだけを書きます`
- `ニュース見出しとチャートの初動を、同じ紙に並べます`

悪い締め:

- `焦って決めない夜にします`
- `明日の自分への貯金みたいです`
- `同じ方向を向くかをメモします`
- `どの材料に資金が反応したかを1行だけメモします`

## 急上昇キーワードの使い方

急上昇キーワードは、相場に無理やり結びつけない。
昼投稿では、読者の行動や成長につながる入口としてだけ使う。
意味のない食べ物ネタや雑談で終わらせない。

例:

```text
＼お昼のひとこと／

急上昇で「梅雨入り」を見て、
相場にも湿度が高い日ってあるなと思いました。

こういう日は、
方向を決めるより
板の薄さを見ていた方がいいかも

予定も荷物も、
大きすぎると持ち運びが大変です

投資家Fより
```

## 投稿が似ないようにするルール

- 直近14日で使ったテーマは避ける
- 朝は相場、昼は日常、夕は行動に寄せる
- `ドル円/GOLD/NASDAQ` を毎回3つ並べない
- `板の薄さ` を毎投稿で使わない
- 絵文字は各投稿3〜6個
- 署名は必ず `投資家Fより💌`
- `こんにちはFです` は使わない
- 朝投稿は注目ニュースを✅で3つ箇条書きする
- 朝投稿はインプレッションを狙える固有名詞・数字・意外性を優先する
- 昼と夕のタイトルを固定しない
- `今日もよろしくお願いします🥰` を朝・昼で自然に使う
- どこかにFの自虐やかわいい比喩を1つ入れる
- 昼投稿は、読者が今日できる小さい行動を必ず1つ入れる
- 昼投稿は、見た人のエネルギーや行動を促す声かけにする
- 意味のないランチ、味の素、眠いだけの投稿は禁止
- 投資助言、売買指示、F自身のポジションは書かない
- `指値` `ポジション` `ロット` `エントリー` `利確` `損切り` `アラート` は使わない
- 締めは抽象的にしない
- 締めには、時間・対象・手元の動作を入れる

## 自然さチェック

投稿前に以下を確認する。

- 声に出して読んで不自然ではないか
- 1投稿1テーマになっているか
- 読者に説教していないか
- AIっぽいまとめで終わっていないか
- 相場用語だけで固くなっていないか
- 絵文字が入っているか
- 朝が注目ニュース3選になっているか
- 朝の✅に固有名詞や数字が入っているか
- 昼と夕のタイトルが固定化していないか
- 昼投稿に「読者が今日できる行動」があるか
- 昼投稿が、読んだ人の背中を押す内容になっているか
- F自身のポジションや売買予定を書いていないか
- 投資助言に見える言葉が入っていないか
- 最後の2行が「何をするか」まで見えるか
- `メモします` だけで終わっていないか
