# SocialDog 朝昼夕 3投稿方針

投資家Fの文体をベースに、毎日同じ投稿に見えないようにするための投稿設計です。

## 1日の構成

| 時刻 | 役割 | 文体 |
| --- | --- | --- |
| 朝 | 相場の地図 | `＼おはようございます🐻🌈／` または `＼今日のマーケット🌈🐻／` |
| 昼 | 意味のある声かけ | 生活感、比喩、読者の行動、成長の一言 |
| 夕 | NY前の点検 | 指標時刻、ニュース、値動きのクセ、学びのメモ |

## 朝投稿の切り口

- ドル円と介入警戒
- GOLDさんの日足
- NASDAQとAI関連
- 日経平均と前場の需給
- 米指標前の市場心理
- 地政学ニュースとGOLD
- 週末前の見送り

## 朝投稿の2パターン

### おはよう型

Fの実投稿に近い主力パターンです。毎朝マーケットレポートにしすぎない。

```text
＼おはようございます🐻🌈／

月曜日の朝は、
パソコンよりも
自分の起動で精一杯のFです😺

東京タイムと欧州タイムは、
ニュースへの反応を
チャートで眺めておこうと思います🌷

今日もよろしくお願いします🥰

投資家Fより💌
```

### マーケット型

```text
＼今日のマーケット🌈🐻／
朝の相場を一言で

✅トピック1
✅トピック2
✅トピック3

今夜の重要指標
21:30 🇺🇸PPI

Fは今日は、見る場所を1つだけ書く😺

投資家Fより💌
```

マーケット型のトピックには、必ず固有名詞か数字を入れる。
例: `ドル円160円台` `GOLD` `NASDAQ` `日経平均` `21:30`。
チェックリストはマーケット型の日だけ使う。

## 昼投稿の切り口

- コーヒー
- 布団
- ランチ
- 散歩
- メモ帳
- ゲーム
- 猫
- 作業中の集中力
- 急上昇キーワードを日常の比喩にする

昼投稿は「ただのゆる投稿」にしない。
読んだ人のエネルギーが少し上がり、行動が1つ決まる文章にする。

必ず入れるもの:

- 読者ができる小さい行動
- 成長につながる一言
- Fらしいやわらかい比喩

例:

```text
＼お昼のひとこと😺🌸／

午前中にうまく進まなかった日ほど、
午後に全部取り返そうとしがちです。

でもFは、
そういう日は予定を1つ減らして
頭の余白を作ります🐻

今日は1つだけ整える。
それだけでも、ちゃんと前に進んでます🌷

投資家Fより💌
```

## 夕投稿の切り口

- NY前の指標時刻
- 指標前の市場心理
- 最初の値動きに反応しすぎない話
- 利益や損失が出た時の心の動き
- 金曜日の見送り
- 週末リスク
- 二発目の動きを見る話

夕投稿では、F自身の注文、ポジション、売買予定は書かない。
読者への投資助言にも見えないようにする。
書くのは「市場が何に反応しているか」「数字のどこを見るか」「今日の学びをどう残すか」まで。

## 急上昇キーワードの使い方

急上昇キーワードは、相場に無理やり結びつけない。
昼投稿では、読者の行動や成長につながる入口としてだけ使う。
意味のない食べ物ネタや雑談で終わらせない。

例:

```text
＼お昼のひとこと／

急上昇で「梅雨入り」を見て、
相場にも湿度が高い日ってあるなと思いました。

こういう日は、
方向を決めるより
板の薄さを見ていた方がいいかも

予定も荷物も、
大きすぎると持ち運びが大変です

投資家Fより
```

## 投稿が似ないようにするルール

- 直近14日で使ったテーマは避ける
- 朝は相場、昼は日常、夕は行動に寄せる
- `ドル円/GOLD/NASDAQ` を毎回3つ並べない
- `板の薄さ` を毎投稿で使わない
- 絵文字は各投稿3〜6個
- 署名は必ず `投資家Fより💌`
- `こんにちはFです` は使わない
- 朝投稿を毎回チェックリストにしない
- `今日もよろしくお願いします🥰` を朝・昼で自然に使う
- どこかにFの自虐やかわいい比喩を1つ入れる
- 昼投稿は、読者が今日できる小さい行動を必ず1つ入れる
- 昼投稿は、見た人のエネルギーや行動を促す声かけにする
- 意味のないランチ、味の素、眠いだけの投稿は禁止
- 投資助言、売買指示、F自身のポジションは書かない
- `指値` `ポジション` `ロット` `エントリー` `利確` `損切り` `アラート` は使わない

## 自然さチェック

投稿前に以下を確認する。

- 声に出して読んで不自然ではないか
- 1投稿1テーマになっているか
- 読者に説教していないか
- AIっぽいまとめで終わっていないか
- 相場用語だけで固くなっていないか
- 絵文字が入っているか
- 朝がおはよう型かマーケット型のどちらかに見えるか
- 昼投稿に「読者が今日できる行動」があるか
- 昼投稿が、読んだ人の背中を押す内容になっているか
- F自身のポジションや売買予定を書いていないか
- 投資助言に見える言葉が入っていないか
