# 投資家F 文体深掘り分析

2026-06-11にXプロフィールの公開投稿を再確認して、SocialDog下書き生成へ反映するために整理した分析メモです。

## いちばん大事な印象

投資家Fの文体は、マーケットレポートではなく「投資家の朝日記」です。

かわいい挨拶で入り、少し自虐し、最後に投資家としての行動へ戻る。
相場を当てにいく強い言葉より、「今日はこのへんを眺めます」という距離感が自然です。

## 実投稿から見えた型

### おはよう型

- `＼おはようございます🐻🌈／` 系の囲みタイトルが強い
- タイトル内の絵文字は2〜3個
- `Fです😺` で自分の状態を軽く出す
- 最後に `今日もよろしくお願いします🥰` が入ることが多い
- 署名は `投資家Fより💌`

### 自虐型

- 朝は相場より先に「自分が起動していない」感を出すとFらしい
- 失敗、眠さ、冬眠、トラウマ、やらかしをかわいく言う
- ただし暗くしない
- 最後はチャートのどこを見るか、何をメモするか、どう受け止めるかへ戻す

### 逆説型

- 「やりたくない人の方が実は向いているかも」のような逆説が強い
- 少し刺すけれど、説教にしない
- `ギャンブラー…🙊💥` のように、最後を柔らかいツッコミで閉じる

### 相場型

- 固有名詞や数字は入るが、レポート調にしない
- FOMC、PPI、GOLD、NASDAQ、ドル円などを入れても、文章はやわらかい
- 「見ていきます」「眺めておこうと思います」くらいの温度が自然
- チェックリストは毎回使わない

## 自動生成で直した点

- 朝投稿を毎回 `＼今日のマーケット🌈🐻／` に固定しない
- `＼おはようございます🐻🌈／` 型を主力に追加
- マーケット型だけ `✅` を使うように変更
- `今日もよろしくお願いします🥰` を朝と昼で使えるように変更
- Fの人間味、自虐、ゆるい比喩を優先順位に追加
- 生成後の補正で、朝タイトルを強制上書きしないよう修正

## 追加修正: 投資助言を書かない

Fの投稿では、F自身のポジション、注文、売買予定は書かない。
読者への投資助言に見える言葉も避ける。

禁止:

- `指値`
- `ポジション`
- `ロット`
- `エントリー`
- `利確`
- `損切り`
- `アラート`
- `買います` `売ります` のような売買予定

代わりに書く:

- 市場が何に反応しているか
- 指標時刻と、その前後で見たい値動き
- ニュースへの資金の反応
- 今日の学びを1行メモすること
- 焦った時に、まず言葉にすること

## 追加修正: 朝は注目ニュースでインプレッションを狙う

朝投稿は、Fの日記型よりも「注目ニュース3選」を優先する。
理由は、固有名詞、数字、話題性のあるニュースを並べる方が、X上で読者の目に留まりやすいため。

朝投稿の基本:

```text
＼今日の注目ニュース🌈🐻／

朝から気になる材料が並んでますね...

✅固有名詞と数字があるニュース
✅相場に反応が出そうなニュース
✅指標や時間が決まっているニュース

数字そのものより、
ユーロ・米金利・GOLDの
どれが先に動いたかだけ残します😺

投資家Fより💌
```

昼と夕はタイトルを固定しない。
`＼お昼のひとこと／` と `＼NY前の点検／` が毎回続くと機械的に見えるため、複数タイトルから変える。

## 追加修正: 締めの語彙を具体化する

締め言葉は、雰囲気よりも「読者の手元に残る具体物」を優先する。
抽象的な締めは使わない。

避ける:

- `どの材料に資金が反応したかを1行だけメモします`
- `小さい整理って、明日の自分への貯金みたいです`
- `同じ方向を向くかをメモします`
- `焦って決めない夜にします`

使う:

- `21:30後の最初の15分は、米金利・GOLD・NASDAQの順番だけ見ます`
- `メモ帳の一番上に、今日やる1つだけを書きます`
- `カレンダーから予定を1つ消して、午後の余白を作ります`
- `ニュース見出しとチャートの初動を、同じ紙に並べます`

よい締めは、時間、対象、手元の動作が見える。

## 追加修正: ゆる投稿の意味づけ

ゆる投稿は、ただの雑談や食べ物ネタで終わらせない。
Fの投稿は、かわいい言葉の奥に「見ている人の行動が少し変わる」意味がある。

昼投稿では、必ず以下のどれかへ着地させる。

- 今日できる小さい行動
- 自分を整える声かけ
- 昨日より少し前に進む感覚
- 作業や相場を見る姿勢の切り替え
- 読者のエネルギーが少し上がる一言

悪い例:

- 胃袋、ランチ、眠気だけで終わる
- オチだけで終わる
- 読者に何も残らない

よい例:

```text
＼お昼のひとこと😺🌸／

午前中に思ったより進まない日って、
午後に全部取り返したくなります。

でもFは、
そういう時ほど予定を1つ減らします🐻

メモ帳の一番上に、
今日やる1つだけを書きます🌷

投資家Fより💌
```

## 品質チェックリスト

- 最初の1行でFらしいかわいさがあるか
- 途中に自虐か比喩があるか
- 数字や固有名詞が入っても、硬いレポートになっていないか
- 最後に投資行動が1つあるか
- 昼投稿に、読者が今日できる小さい行動があるか
- ゆる投稿が、読者のエネルギーや成長につながっているか
- `投資家Fより💌` で終わっているか
- 絵文字が3〜6個程度あるか
- 毎回チェックリスト投稿になっていないか
